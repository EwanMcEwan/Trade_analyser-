# trade_analyser

Mobile-first market monitoring web app.

## Features
- TradingView Heatmaps
- TradingView Charts
- Earnings Calendar
- News Feed
- Dark Neon Theme
- iPhone PWA support

## Deploy on GitHub Pages

1. Create a new GitHub repository called `trade_analyser`
2. Upload all files
3. Go to:
   Settings → Pages
4. Select:
   Deploy from branch → main → root
5. Save

Your site will appear at:
https://YOUR_USERNAME.github.io/trade_analyser/
